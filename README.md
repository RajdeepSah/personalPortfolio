## ğŸš€ Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone <your-repo-url>
cd portfolio
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## ğŸ¨ Customization

### Personal Information

Update the following files with your personal information:

1. **Hero Section** (`src/components/Hero.tsx`):
   - Update name, title, and tagline
   - Update social media links
   - Update resume download link

2. **About Section** (`src/components/About.tsx`):
   - Update education timeline
   - Update career milestones
   - Update skills and competencies

3. **Projects Section** (`src/components/Projects.tsx`):
   - Add your projects with descriptions
   - Update GitHub and demo links
   - Add relevant tech stacks

4. **Experience Section** (`src/components/Experience.tsx`):
   - Update professional experience
   - Add internships and leadership roles
   - Update achievements and skills

5. **Research Section** (`src/components/Research.tsx`):
   - Add your publications
   - Update research areas
   - Add upcoming research projects

6. **Contact Section** (`src/components/Contact.tsx`):
   - Update contact information
   - Update social media links
   - Configure contact form (if using backend)

### SEO and Meta Tags

Update the metadata in `src/app/layout.tsx`:
- Title and description
- Open Graph tags
- Twitter card information
- Keywords

### Styling

The website uses TailwindCSS for styling. You can:
- Modify colors in the components
- Update the mathematical background animation
- Customize animations and transitions
- Add new sections or modify existing ones
